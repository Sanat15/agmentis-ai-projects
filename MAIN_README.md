# Agmentis.ai Projects

This repository contains two projects built during the selection process and internship evaluation at **Agmentis.ai**.

---

## Projects

### 1. Smart Document Program
> Semantic search prototype for PDF documents using natural language queries

A RAG-based document intelligence system that allows users to ask natural language questions over uploaded PDF documents. Built using Python, LangChain, and vector embeddings.

📁 [`/smart-document-program`](./smart-document-program)

---

### 2. WebRTC Browser Extension
> Chrome extension with live tab audio capture and WebRTC calling

A Chrome browser extension that demonstrates two things — capturing live audio from any browser tab using `chrome.tabCapture`, and making real peer-to-peer WebRTC audio calls directly from the extension. Includes a Node.js signalling server.

📁 [`/webrtc-extension`](./webrtc-extension)
